Toast Plugin for Bootstrap 4.6

This is a guide on how to install and use the Toast Plugin for Bootstrap 4.6 in your project.

Installation Guide

Follow these steps to install and set up the Toast Plugin:

  1. Download or copy the showToast function into a JavaScript file (e.g., toast-plugin.js) in your project.
  2. Make sure you include Bootstrap CSS and JS in your HTML file.

Include Bootstrap CSS and JS

    
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.6.2/css/bootstrap.min.css">
    <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.6.2/js/bootstrap.bundle.min.js"></script>

Link the Toast Plugin JavaScript

    
    <script src="path/to/your/toast-plugin.js"></script>

Ensure your icons (e.g., tick.png, error.png) are placed in an assets folder relative to the JavaScript file.

Usage Guide

Once installed, you can use the showToast function to display different types of toasts in your application.

Basic Example

    
    showToast('Operation completed successfully!', {
        title: 'success',
        delay: 3000,
        smallText: 'Just now',
    });

Customizing the Toast

You can customize the following options:

  • title: The title of the toast. Examples: success, error, info.
  • delay: The delay in milliseconds before the toast hides. Default is 2500ms.
  • smallText: The subtitle (e.g., a timestamp). Default is 'Just now'.

Advanced Examples

Error Toast

    
    showToast('An unexpected error occurred.', {
        title: 'error',
        delay: 4000,
        smallText: 'A moment ago',
    });

Warning Toast

    
    showToast('Please check your input fields.', {
        title: 'warning',
        delay: 3000,
        smallText: '2 minutes ago',
    });

Directory Structure

Ensure your project follows the correct folder structure for the plugin to work correctly:

    
    project/
    ├── index.html
    ├── assets/
    │   ├── tick.png
    │   ├── error.png
    │   ├── warning.png
    │   ├── info.png
    └── js/
        └── toast-plugin.js
